# 5 вариантов тёмного дизайна — Дневник трейдера

Каждый файл — самостоятельный HTML-мокап дашборда. Открой `index.html` в браузере для галереи всех вариантов.

## Варианты

| # | Файл | Концепция | Палитра | Подойдёт если… |
|---|------|-----------|---------|----------------|
| 01 | `01_obsidian_minimal.html` | Чистый минимализм. Острые края, тонкие линии, без скруглений и теней. Типографика и сетка решают всё. | Чёрный + белый + графит, акценты — зелёный/красный для P&L | хочется ощущение «дорого и тихо», без визуального шума |
| 02 | `02_midnight_glass.html` | Тёмный glassmorphism. Размытые панели, мягкие подсветки от ауры в фоне, спарклайны. | Глубокий тёмно-синий + cyan/purple + glass-blur | нужна современная «премиум-SaaS» эстетика |
| 03 | `03_bloomberg_terminal.html` | Финансовый терминал. Моноширинный шрифт, янтарные акценты, командная строка, F-клавиши, таблицы. | Чёрный + amber #ffb000 + green/red тикеры | хочется почувствовать себя на торговом столе institutional-трейдера |
| 04 | `04_slate_professional.html` | Премиум-сланец. Serif-заголовки (Playfair), золотые акценты, мягкие тени, аккуратные карточки. | Slate-900 + золото #c8a872 | дневник для «серьёзного» трейдера, акцент на статус и читаемость |
| 05 | `05_tokyo_night.html` | Tokyo Night IDE. Vibe редактора кода — табы, моноширинный, цветные акценты, командные шорткаты. | #1a1b26 + #7aa2f7 + cyan/purple/orange | если ты любишь VS Code / Neovim и хочешь чтобы дневник «выглядел как репозиторий» |

## Как использовать

1. Открой `index.html` чтобы увидеть превью всех 5 вариантов рядом (iframe-галерея).
2. Кликни заголовок варианта — откроется полный мокап в новой вкладке.
3. Когда выберешь — перенесём цветовую систему и компоненты в реальные шаблоны Django (`base.html` + `core/css/main.css`).

## Что в каждом мокапе

Один и тот же контент дашборда (для честного сравнения):
- Навигация + сайдбар с пунктами меню
- 4 KPI-карточки (Всего сделок, Закрытых, Открытых, P&L)
- Панель «Последние сделки» (5 строк)
- Панель «Активные стратегии» (3 строки)
- Быстрые действия

Различия — только в визуальной системе: палитре, типографике, форме, плотности, акцентах.
