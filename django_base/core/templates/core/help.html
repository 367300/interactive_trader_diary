Часто задаваемые вопросы

Для добавления новой сделки:

  1. Перейдите в раздел "Сделки" или нажмите кнопку "Новая сделка" на дашборде
  2. Заполните обязательные поля: инструмент, дата/время, направление, цена входа, количество
  3. Укажите стратегию, если она есть
  4. Добавьте анализ сделки и основание для входа
  5. Нажмите "Сохранить"

Для настройки депозита:

  1. Перейдите в раздел "Профиль" в верхнем меню
  2. Укажите начальный депозит или текущий депозит
  3. Можете включить опцию "Использовать дефолтный депозит (1 млн руб.)"
  4. Нажмите "Сохранить"

Депозит используется для расчета процента доходности и других метрик.

Для создания торговой стратегии:

  1. Перейдите в раздел "Стратегии"
  2. Нажмите "Создать стратегию"
  3. Укажите название и описание стратегии
  4. Выберите тип стратегии (скальпинг, дневная торговля, свинг, позиционная)
  5. Укажите типы инструментов (акции, фьючерсы или оба)
  6. Нажмите "Сохранить"

Для закрытия сделки:

  1. Найдите нужную сделку в списке сделок
  2. Нажмите "Редактировать"
  3. Укажите цену выхода
  4. Рассчитайте и укажите фактический результат
  5. Отметьте галочку "Сделка закрыта"
  6. Добавьте основание для закрытия в анализе
  7. Нажмите "Сохранить"

Win Rate — это процент прибыльных сделок от общего количества закрытых сделок.

Формула: (Количество прибыльных сделок / Общее количество закрытых сделок) × 100%

Win Rate показывает, как часто ваши сделки оказываются прибыльными. Высокий win rate не всегда означает прибыльность — важнее общий P&L.

Руководство пользователя

Начало работы
  1. Зарегистрируйтесь или войдите в систему
  2. Настройте профиль и депозит
  3. Создайте торговые стратегии
  4. Начните добавлять сделки
Эффективное использование
  • Регулярно ведите дневник сделок
  • Анализируйте свои эмоции и ошибки
  • Используйте теги для группировки сделок
  • Периодически просматривайте статистику
